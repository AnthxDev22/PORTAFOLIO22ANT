Sobre Mí

Soy un apasionado del desarrollo y la infraestructura de Minecraft.

Mi camino en el mundo de Minecraft Java comenzó como el de muchos: jugando y administrando servidores comunitarios con amigos. Al ver los desafíos técnicos que implicaba mantener una comunidad activa y libre de lag, decidí ir más allá. Así fue como empecé a sumergirme en el código del servidor: analizando configuraciones, entendiendo la lógica de los archivos .yml y estudiando Java para comprender exactamente cómo interactúan los plugins.

Hoy en día, combino mis conocimientos de programación con una sólida experiencia en optimización e integración. Más que crear herramientas desde cero, mi enfoque principal es hacer que la infraestructura existente sea extremadamente estable y eficiente para garantizar una experiencia de juego superior bajo cualquier nivel de carga.

Mi Arsenal Técnico

  • Software de Servidor: PaperMC, Spigot, Purpur, Velocity, BungeeCord.
  • Bases de Datos & Sincronización: MySQL, SQLite, MongoDB, Redis (Persistencia global de datos de usuarios).
  • Diagnóstico de Rendimiento: Spark, Timings v2, Optimización de Java Garbage Collection (Aikar’s Flags).
  • Gestión de Plugins Core: Configuración avanzada de LuckPerms, DeluxeMenus (GUIs dinámicas), PlaceholderAPI.
  • Herramientas de Entorno: Git, GitHub, lectura de Stacktraces/Consola en Java.

Mi Enfoque en Proyectos

1

Mitigación de Lag

2

3

Interfaces Limpias

Creación de menús interactivos complejos optimizados para no saturar el hilo principal del servidor.
